# 🚀 Kubernetes Hands-On Practical Session

## 📘 Topics Covered

- Kubernetes Architecture
- Pods
- Namespaces
- Deployments
- StatefulSets
- Services
- ConfigMaps
- Secrets
- Persistent Volumes (PV)
- Persistent Volume Claims (PVC)

---

# 🛠️ Prerequisites

- Docker Installed
- kubectl Installed
- Minikube Installed

---

# 🚀 Start Cluster

```bash
minikube start
```

---

# 🔍 Verify Cluster

```bash
kubectl get nodes
kubectl cluster-info
```

---

# 🧹 Cleanup

```bash
kubectl delete -f .
```
